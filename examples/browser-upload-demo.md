# Browser Upload Demo

## How to Activate a Skill in Any Browser LLM
1. Open your preferred web-based AI interface (ChatGPT, Claude, Gemini, Hugging Face, etc.)
2. Start a new chat session
3. Paste the entire contents of a `SKILL.md` file into the prompt box
4. Add a clear separator and your task:

[PASTED SKILL.md CONTENTS]

TASK: Clean this CSV data and return a JSON summary:
id,name,date,amount
1,Alice,01/15/2024,250.00
2,Bob,2/3/24,180
3,,01/2024,300
